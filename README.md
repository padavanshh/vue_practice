# project

## Project setup
```
yarn install
```
### You must have json installed
```
yarn global add json-server
```
### Start the json-server
```
json-server data.json
```
### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
